# 🌦️ Application Météo - Réseau de Transport

## 📌 Description
Ce projet est une interface de visualisation météo en temps réel destinée aux écrans d'information dans les transports en commun. Il a été réalisé dans le cadre d'une candidature pour la formation **Concepteur Développeur d'Applications (CDA)**.



## ✨ Fonctionnalités
- Récupération des données météo en temps réel via l'API **OpenWeatherMap**.
- Configuration dynamique de la ville via un fichier local `conf.json`.
- Affichage automatique : Température (arrondie), Humidité, Vitesse du vent.
- Mise à jour automatique des données **toutes les heures** (sans rechargement de la page).

## 🛠️ Technologies utilisées
- **HTML5 / CSS3** : Structure et mise en page.
- **JavaScript (ES6+)** : Logique asynchrone avec `fetch` et manipulation du DOM.
- **JSON** : Gestion de la configuration locale.
